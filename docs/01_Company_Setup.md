# Step 1: Company Setup in SAP HR

## Objective
Create a fictitious company to demonstrate the H2R process.

## Company Details
- **Company Name:** TechNova Pvt. Ltd.
- **Industry:** Information Technology
- **Location:** Noida, India
- **Employees:** 500+

## SAP Configuration Steps

### 1.1 Define Company Code
- **T-Code:** OX02
- Path: IMG → Enterprise Structure → Definition → Financial Accounting → Edit, Copy, Delete, Check Company Code
- **Company Code:** TN01
- **Company Name:** TechNova Pvt. Ltd.
- **Country:** IN
- **Currency:** INR

### 1.2 Assign Company Code to Company
- **T-Code:** OX16
- Link TN01 to parent company TNGP (TechNova Group)
