# Step 3: Personnel Area & Subarea Setup

## Objective
Define where employees work (location-based HR grouping).

## SAP Configuration

### 3.1 Define Personnel Area
- **T-Code:** OX03
- Path: IMG → Enterprise Structure → Definition → Human Resources Management → Personnel Areas
- **Personnel Area:** TN01 (TechNova Noida)
- **Name:** TechNova Noida Office

### 3.2 Define Personnel Subarea
- **T-Code:** OX03 (same, subarea tab)
- **Subarea:** TN01 (IT Wing)

### 3.3 Assign Personnel Area to Company Code
- **T-Code:** OX01
- Assign TN01 Personnel Area → TN01 Company Code

### Employee Groupings
| Group | Description |
|-------|-------------|
| 1 | Active Employees |
| 2 | Probation |
| 3 | Contract Staff |
